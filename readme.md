# EN- Cybersecurity Notes

This repository contains my personal study notes related to cybersecurity.
The content is focused on technical fundamentals, tools, and security techniques
from both offensive (red team) and defensive (blue team) perspectives.

## Purpose

- Organize knowledge from courses, labs and self-study
- Improve technical retention through documentation
- Build a public technical portfolio
- Practice technical writing

## Topics Covered

- Linux and system administration
- Networking fundamentals
- Web technologies
- Reconnaissance and scanning
- Firewall and evasion concepts
- Log analysis and detection
- Security tools (Nmap, Burp Suite, etc.)

## Disclaimer

This repository does not contain:
- Lab solutions
- Exploits targeting real systems
- Course material or proprietary content

All content is written for educational purposes only.

## Status

This repository is continuously updated as part of my cybersecurity learning process.
