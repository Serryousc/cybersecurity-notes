#Linux bases - Bash/Terminal 

##Basic Navegation Commands
  pwd -> Show your current location
  man -> Show how a command works example: man pwd (try your self)
  ls -> list all directories and archives in your current location (List Directory)
      Especial case of use with ls command:
          ls -l -> list all directories and archives in your current location with more details
          ls -a -> list all directories and archives in your current location include hidden files
  observation: all archive that starts with "." is hidden by default
  cd -> change for another directory (Change Directory)
  cd.. -> change for a previous directory
  cd /directory/folden/location -> Change for that exsately directory (location)
